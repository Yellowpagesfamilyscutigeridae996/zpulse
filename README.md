zpulse is a monitoring dashboard for home racks and small server setups. It shows the health of your ZFS pools, disks, and SMART data in one place. It helps you keep track of drives, snapshots, and server status without opening each machine by hand.

Use it when you want to:

- Track SSD, SATA, SAS, and NVMe drives
- See snapshot activity
- Monitor more than one server from one screen
